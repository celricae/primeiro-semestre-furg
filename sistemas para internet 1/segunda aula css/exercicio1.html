CSS Exemplo de Alinhamento

Setembro de 2014

A propriedade text-align faz exatamente o que o nome sugere: alinha o texto da mesma forma que ocorre os editores de texto. Exsitem quatro tripos de alinhamento para textos em HTML com CSS:

  • centralizado (center),
  • justificado (justify),
  • esquerda (left) e
  • direita (right).

PS:Redimensione a janela para ver se o texto justificado funciona.
